# TCMI-F-6D-Benchmark
This repository presents TCMI-F-6D, a benchmark designed to evaluate the foundational interdisciplinary competency of large language models (LLMs) in Traditional Chinese Medicine Informatics (TCMI). Grounded in Cognitive Hierarchy Theory and Disciplinary Knowledge Structure Theory, the benchmark is built from six core disciplines selected from the MMLU dataset and is accompanied by a composite metric system. We use it to assess model baseline capability, learning gains, and performance stability across TCMI-related interdisciplinary scenarios.
